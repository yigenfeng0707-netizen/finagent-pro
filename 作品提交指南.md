# FinAgent Pro 作品提交指南

## 提交前的准备

### 1. 代码仓库准备（推荐）

将项目上传到GitHub或Gitee：

```bash
# 进入项目目录
cd d:\HK_AI_hack\finagent-pro

# 初始化Git仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "FinAgent Pro - 多Agent智能投顾系统"

# 创建GitHub仓库后，添加远程仓库
git remote add origin https://github.com/你的用户名/finagent-pro.git
git branch -M main
git push -u origin main
```

### 2. 打包完整项目

创建zip压缩包：

```
finagent-pro/
├── backend/
│   ├── agents/
│   ├── knowledge/
│   ├── services/
│   ├── main.py
│   ├── requirements.txt
│   └── .env.example
├── frontend/
│   ├── src/
│   ├── public/
│   ├── package.json
│   └── tsconfig.json
├── README.md
├── 演示脚本.md
├── 部署指南.md
├── 演示PPT.pptx
└── 项目验证报告.md
```

## 常见提交方式

### 方式1：在线表单提交（最常见）

大多数hackathon通过以下表单提交：

| 提交项 | 说明 |
|--------|------|
| 项目名称 | FinAgent Pro |
| 项目简介 | 1-2段项目描述 |
| 仓库链接 | GitHub/GitLab地址 |
| 演示视频 | 3-5分钟功能演示 |
| PPT下载链接 | 路演PPT |
| 团队成员 | 姓名、联系方式 |
| 技术栈说明 | 国产大模型 + AKShare + ChromaDB |

### 方式2：邮件提交

发送邮件到指定邮箱：

**邮件主题**: `[HK Physical AI Hackathon] FinAgent Pro - AI Agent赛道`

**邮件内容**:
```
项目名称：FinAgent Pro - 多Agent智能投顾系统
参赛赛道：AI Agent
团队成员：[你的姓名]
联系方式：[邮箱/电话]
仓库地址：[GitHub链接]
项目简介：
  基于国产大模型的港股AI投资助手，通过多Agent协作
  （市场分析师、风险经理、组合顾问、情绪扫描器），
  为投资者提供专业的投资建议和风险评估。

附件：
1. FinAgent_Pro_演示.pptx
2. 项目演示视频链接
```

### 方式3：现场提交

如果需要现场提交：

```
提交清单：
✅ U盘/移动硬盘（项目完整代码）
✅ 打印版项目说明书
✅ 演示PPT（电子版）
✅ 团队成员名片/联系方式
✅ 笔记本电脑（现场演示）
```
