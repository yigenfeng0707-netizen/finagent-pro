## 一、项目概述

### 1.1 一句话介绍

FinAgent Pro 是一个基于国产大模型的多Agent智能投顾系统，通过DAG并行编排四位AI专家（市场分析师、情绪扫描器、风险经理、组合顾问）协同决策，为港股投资者提供45秒端到端的专业级投资分析服务。

### 1.2 解决的核心问题

中国港股市场拥有200万+活跃散户投资者，但专业投顾服务门槛高达50万港币，智能投顾渗透率不足15%。散户面临**信息过载、风险难控、服务不均、黑盒决策、境外依赖**五大痛点，亟需一个零成本、可解释、国产自主的智能投顾解决方案。

### 1.3 解决方案

FinAgent Pro 模拟专业投研团队的工作流程：

- **4位AI专家**各自负责技术面、情绪面、风险面、配置面分析
- **DAG并行编排引擎**让无依赖步骤并行执行，节省40%分析时间
- **8个金融专业工具**包括马科维茨优化、压力测试、VaR/CVaR等
- **WebSocket实时推送**让投资者看到每一步AI思考过程，决策完全可追溯
- **国产技术栈**DeepSeek/GLM-4 + ChromaDB + AKShare，全链路国内网络运行

## 二、技术创新性（评审权重40%）

### 2.1 DAG并行编排引擎

**创新点**：传统多Agent系统采用线性链式执行，FinAgent Pro 创新性地实现了DAG（有向无环图）并行编排引擎。

市场分析与情绪扫描之间无数据依赖，**并行执行**；风险评估同时消费两个并行步骤的输出，实现信息融合；组合建议综合前三步全部结果。相比链式执行节省约40%分析时间。

支持**动态步骤增减**、**失败容错**（单步失败不阻塞后续）、**超时降级**（90s超时自动降级）。

### 2.2 智能意图识别

**创新点**：采用规则快速匹配 + LLM回退增强的双层架构，替换传统硬编码关键词匹配。

第一层规则引擎毫秒级响应，覆盖股票代码/名称/金额/风险偏好等常见场景；第二层LLM增强处理"10万块保守型帮我看看腾讯和阿里怎么配"等复杂自然语言。兼顾效率和覆盖面。

### 2.3 八个金融专业工具

| 工具 | Agent | 能力 |
|------|-------|------|
| get_stock_price | 市场分析师 | 实时股价获取 |
| get_technical_indicators | 市场分析师 | MA/MACD/RSI等技术指标 |
| **get_fundamentals** | 市场分析师 | **PE/PB/市值等基本面指标（新增）** |
| get_fund_flow | 情绪扫描器 | 资金流向分析 |
| get_portfolio_risk | 风险经理 | 协方差矩阵组合风险 |
| calculate_var | 风险经理 | VaR/CVaR/夏普比率 |
| **stress_test** | 风险经理 | **4种极端情景压力测试（新增）** |
| **markowitz_optimize** | 组合顾问 | **马科维茨均值-方差优化（新增）** |

### 2.4 RAG知识库增强

ChromaDB + all-MiniLM-L6-v2 语义搜索，为每个Agent自动注入金融领域知识。支持动态添加/删除/批量导入知识条目（管理API已开放）。

### 2.5 LLM容错降级体系

90s超时 + 3次指数退避重试 + DeepSeek→智谱GLM-4主备切换 + 流式SSE响应 + 单步失败不阻塞DAG。

### 2.6 技术壁垒与竞争优势

| 对比维度 | 传统智能投顾 | CrewAI等通用框架 | **FinAgent Pro** |
|----------|-------------|-----------------|------------------|
| Agent协作 | 单体模型 | 链式/黑盒 | **DAG并行+实时可视化** |
| 决策可解释 | 黑盒 | 黑盒 | **WebSocket流式推送思考过程** |
| 金融专业性 | 通用问答 | 通用框架 | **8个金融专业工具+RAG知识库** |
| 风险评估 | 简单评分 | 无 | **VaR+CVaR+压力测试+马科维茨优化** |
| 国产化 | 依赖OpenAI | 部分依赖 | **全链路国产，零VPN依赖** |
| 运行成本 | API付费 | 框架免费+API付费 | **零成本（开源+免费数据源）** |
