# FinAgent Pro 演示录屏脚本

> 用于录制3分钟演示视频，配合 AFAC2026 参赛材料提交

## 录屏前准备

1. 启动后端: `cd backend && python main.py`
2. 启动前端: `cd frontend && npm start`
3. 打开浏览器访问 http://localhost:3000
4. 准备录屏工具: OBS Studio / 浏览器插件 Loom / Windows Win+G

## 录屏流程（3分钟）

### 0:00-0:15 开场介绍
- 展示首页仪表盘
- 口述: "FinAgent Pro — 基于国产大模型的多Agent智能投顾系统，AFAC2026方向四Agentic AI参赛作品"

### 0:15-0:45 核心功能演示 — 智能分析
- 在仪表盘选择"腾讯控股 00700"
- 点击"开始分析"
- 展示4个Agent并行执行过程（WebSocket实时推送）
- 口述: "市场分析和情绪扫描并行执行，风险评估依赖前两者，DAG编排节省40%时间"

### 0:45-1:15 Agent对话演示
- 切换到"Agent对话"页面
- 输入: "帮我分析腾讯和阿里，10万保守型"
- 展示意图识别结果 + Agent分析过程
- 口述: "规则引擎毫秒级匹配，LLM回退处理复杂自然语言"

### 1:15-1:45 风险评估 & 组合优化
- 切换到"组合分析"页面
- 展示马科维茨优化结果（scipy精确求解）
- 切换到"风险评估"页面
- 展示VaR/CVaR/压力测试结果
- 口述: "学术级风险量化，VaR+CVaR+4种压力情景+相关性聚集调整"

### 1:45-2:15 数字员工工作台
- 切换到"数字员工工作台"
- 展示DAG决策溯源图
- 口述: "每一步推理可追溯，从黑盒到白盒"

### 2:15-2:40 技术亮点总结
- 切换到暗色模式（Header右上角开关）
- 口述总结4大创新:
  1. DAG并行编排引擎
  2. 多因子评分+LLM综合推理
  3. 8个金融专业工具（scipy马科维茨+学术级压力测试）
  4. 全栈国产化（DeepSeek+智谱GLM+AKShare）

### 2:40-3:00 结尾
- 口述: "FinAgent Pro — 让AI成为每个投资者的专业顾问"
- 展示GitHub仓库页面

## 录屏设置建议

- 分辨率: 1920x1080
- 帧率: 30fps
- 格式: MP4 (H.264)
- 文件大小目标: < 50MB
- 浏览器缩放: 100%

## 导出后

将录屏文件命名为: `finagent-pro-afac2026-demo.mp4`
放置于项目根目录，并在大赛平台"产品原型外链"处上传。
