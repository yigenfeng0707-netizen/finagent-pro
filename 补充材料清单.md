## 四、修改建议提交方式

大赛平台提交后，可在大赛官网 **afac.alipay.com** 的个人中心找到"修改作品"入口进行更新（截止前2次修改机会）。

### 具体步骤
1. 浏览器打开 **https://afac.alipay.com**
2. 登录后进入 **个人中心 → 我的赛事 → AFAC2026**
3. 找到 **"修改作品"** 按钮（通常在作品提交时间线附近）
4. 在弹出的窗口/页面中：
   - 1️⃣ 商业计划书 → 上传 `FinAgentPro商业计划书.docx`
   - 2a → 填写GitHub/Demo链接
   - 2b → 视频已是最新（如果录屏需要更新，请重新生成）
   - 2c → 上传 `FinAgentPro交互设计与流程图.docx`
   - 2d → 上传 `FinAgentPro_MVP_POC实验数据.docx`
5. 确认提交
6. 在 **补充材料** 区域上传本目录下其他 PDF/DOCX 文档

## 五、注意事项

- 大赛平台支持 docx/pdf/mp4/png 等格式
- 单个文件大小限制通常为 50MB（具体看平台规定）
- 修改截止前会保留最后一次提交版本作为最终参赛作品
- 修改不影响已通过的资格审核
